# task12_data_transformation

This project focuses on cleaning and transforming an HR Analytics dataset
using Power BI Power Query and building basic analytical visuals.

## Tools Used
- Power BI Desktop
- Power Query

## Key Steps
- Imported HR Analytics dataset
- Checked column quality (no missing values found)
- Removed unnecessary columns and renamed headers
- Verified and corrected data types
- Created Age Group and Salary Band using conditional columns
- Built visuals for analysis

## Output
- Power BI dashboard with multiple visuals
- Exported dashboard in PDF format
